Image search abstraction miscroservice

Microservice that provides image search result details.

Service uses Google image search API to get search results. Returns the following information for each image:

  • URL of image
  • Snippt (short description of image)
  • URL of thumbnail
  • Context URL

Search

To do an image search, enter URL into web address bar as per format below. When entering address, replace [SEARCH-TERM] with the term to be searched.

http://[APP ADDRESS].herokuapp.com/api/[SEARCH-TERM]?offset=1

Adjusting the value of offset will change which page is displayed (default is 1). Note that 10 results are displayed per page.


History

To view recent search history, enter URL into web address bar as per format below.

http://[APP ADDRESS].herokuapp.com/history?num=10

Adjusting value of num will change number of recent search terms to be displayed (default is 10). Note that maximum recent search results displayed is 100.
